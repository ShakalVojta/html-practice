How do I see the differences between HTML and CSS?

  1. What is the difference between the usage of HTML and CSS?
    • HTML is used for structuring content on a webpage, while CSS is used for styling and formatting that content.
  2. What is the newest version of HTML?
    • HTML5 was the latest version of HTML.
  3. What are the top five most used HTML5 elements? If not sure, make a guess.
    • div: General-purpose container element.
    • a: Anchor element for creating hyperlinks.
    • img: Image element for embedding images.
    • p: Paragraph element for defining paragraphs.
    • h1 to h6: Heading elements for defining headings of different levels.
  4. Have you checked out the HTML5 cheatsheet?
    • Yes, I looked and really enjoyed it.
  5. What is the difference between semantic and non-semantic elements?
    • Semantic elements convey meaning to both browsers and developers, aiding in document structure and accessibility, while non-semantic elements are generic containers without conveying specific meaning, often used for styling purposes.
  6. Why do we prefer using semantic elements?
    • We prefer using semantic elements because they provide clear and meaningful structure to web documents, enhancing accessibility, search engine optimization, and facilitating easier understanding by both developers and browsers.
  7. Do all browsers support all of the newest HTML / CSS elements?
    • Major modern browsers like Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge generally support the newest HTML and CSS elements. However, older versions of Internet Explorer may lack full support for some features. For instance, Internet Explorer 11 might have limited compatibility compared to more modern browsers.
